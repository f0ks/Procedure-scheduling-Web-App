# Procedure-scheduling-Web-App

**Run the server:**
1. Install Node.js and MongoDB, run `mongod`

2. Run following commands in terminal:

`cd server`

`npm i`

`node server.js`

**Run the front-end Angular app:**
1. Install Angular CLI:
`npm install -g @angular/cli`

2. Run following commands in terminal:

`cd front-end`

`npm i`

`ng serve --open`
